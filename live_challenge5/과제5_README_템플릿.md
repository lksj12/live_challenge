## 5. 실행 방법

### 5-1. 설치 및 실행

```bash
npm install
npm run dev
```

브라우저 기반 환경을 사용한 경우:

```text
실행 환경: StackBlitz / CodeSandbox / GitHub Codespaces / 기타
실행 링크: [링크 작성]
```

### 5-2. 실행 확인 방법

1. 브라우저에서 실행 주소를 엽니다.
2. 메인 화면에 목록 또는 빈 상태 메시지가 보이는지 확인합니다.
3. 새 항목을 추가합니다.
4. 기존 항목을 수정합니다.
5. 항목을 삭제합니다.
6. LocalStorage를 사용한 경우 새로고침 후 데이터가 유지되는지 확인합니다.
7. Supabase를 사용한 경우 DB 반영 여부와 RLS/권한 설정을 확인합니다.

2. 현재 선택된 셀 좌표 표시 기능

화면 상단에 현재 선택된 셀의 좌표를 표시하는 영역을 만들어 주세요.

초기 상태는 다음처럼 표시합니다.

Cell: -

사용자가 셀을 클릭하거나 focus하면 다음 형식으로 표시해야 합니다.

Cell: D4

예시:

* C열 1행을 선택하면 Cell: C1
* D열 4행을 선택하면 Cell: D4
* A열 2행을 선택하면 Cell: A2

행과 열 index는 내부적으로 0부터 시작해도 되지만, 화면에는 실제 스프레드시트처럼 표시해야 합니다.

3. 행/열 헤더 하이라이트 기능

사용자가 셀을 선택하면 다음 요소를 함께 하이라이트해 주세요.

* 현재 선택된 셀의 열 헤더
* 현재 선택된 셀의 행 헤더
* 현재 선택된 셀 자체

예를 들어 D4 셀을 선택하면:

* D 열 헤더가 하늘색으로 표시되어야 합니다.
* 4 행 헤더가 하늘색으로 표시되어야 합니다.
* D4 셀 자체도 배경색이나 테두리로 구분되어야 합니다.

새로운 셀을 선택하면 이전 하이라이트는 제거되고, 새로 선택된 셀에만 하이라이트가 적용되어야 합니다.

4. 셀 데이터 입력 및 유지

각 셀은 독립적인 값을 가져야 합니다.

예를 들어 사용자가 다음처럼 입력할 수 있어야 합니다.

* A2에 3333
* B3에 222
* D4에 ㅇ의

다른 셀을 클릭하거나 이동해도 기존 값은 유지되어야 합니다.

또한 브라우저를 새로고침해도 입력한 값이 유지되어야 합니다.
이를 위해 localStorage를 사용해 주세요.

저장해야 하는 정보는 다음과 같습니다.

* 전체 셀 데이터
* 현재 행 개수
* 현재 열 개수

저장된 데이터가 없으면 기본 9행 × 9열 빈 데이터를 생성해 주세요.
저장된 데이터가 손상되어 JSON 파싱에 실패하면 기본 9행 × 9열 상태로 초기화해 주세요.

* C3 → data[2][2]
* D4 → data[3][3]

빈 셀은 반드시 빈 문자열 ""로 저장해 주세요.

12. SheetJS 사용 요구사항

Excel 및 CSV 파일 생성을 위해 SheetJS를 사용해 주세요.

index.html에는 반드시 아래 CDN을 포함해 주세요.

<script src="https://cdn.jsdelivr.net/npm/xlsx/dist/xlsx.full.min.js"></script>

.xlsx 저장 시 요구사항:

* XLSX.utils.aoa_to_sheet()를 사용해서 2차원 배열을 worksheet로 변환합니다.
* XLSX.utils.book_new()로 workbook을 생성합니다.
* XLSX.utils.book_append_sheet()로 Sheet1 이름의 시트를 추가합니다.
* XLSX.write()를 사용해 ArrayBuffer를 생성합니다.
* fileHandle.createWritable()을 사용해 선택한 위치에 저장합니다.

.csv 저장 시 요구사항:

* 2차원 배열을 worksheet로 변환합니다.
* XLSX.utils.sheet_to_csv()를 사용해 CSV 문자열로 변환합니다.
* CSV 저장 시 UTF-8 BOM을 추가해서 한글이 깨질 가능성을 줄여 주세요.
* fileHandle.createWritable()을 사용해 선택한 위치에 저장합니다.

19. 테스트 기준

구현 후 다음 동작이 가능해야 합니다.

1. 페이지 접속 시 기본 9×9 스프레드시트가 표시된다.
2. 열 헤더는 A~I로 표시된다.
3. 행 헤더는 1~9로 표시된다.
4. C1 셀을 클릭하면 Cell: C1이 표시된다.
5. D4 셀을 클릭하면 D 열 헤더와 4 행 헤더가 하늘색으로 하이라이트된다.
6. A2에 3333을 입력한 뒤 다른 셀을 클릭해도 값이 유지된다.
7. D4에 한글 ㅇ의를 입력해도 값이 유지된다.
8. 새로고침 후에도 입력한 값이 유지된다.
9. Add Row 버튼을 누르면 행이 1개 추가된다.
10. Add Column 버튼을 누르면 열이 1개 추가된다.
11. 특정 셀을 선택한 뒤 Delete Row를 누르면 해당 행이 삭제된다.
12. 특정 셀을 선택한 뒤 Delete Column을 누르면 해당 열이 삭제된다.
13. 행/열 추가 또는 삭제 후 새로고침해도 변경된 행/열 개수가 유지된다.
14. Export SpreadSheet 버튼을 누르면 로컬 파일 저장 창이 열린다.
15. 저장 창에서 파일명, 저장 위치, .xlsx 또는 .csv 형식을 선택할 수 있다.
16. .xlsx 파일이 정상 저장된다.
17. .csv 파일이 정상 저장된다.
18. 저장된 파일을 Google Spreadsheet에서 Import하면 웹앱에서 입력한 데이터와 같은 셀 위치에 표시된다.
